Olympics 2040

Process Summary

This project involved creating a theoretical brand identity and logo for the 2040 Summer Olympics, assuming the event was hosted in Kyoto, Japan. I explored multiple logo concepts before selecting a final design inspired by traditional torii gates found throughout the country. The project also included designing a set of sport iconography for select Olympic events and prototyping event tickets that attendees would use during the Games.
