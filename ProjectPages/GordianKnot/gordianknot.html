The Gordian Knot

Process Summary

The Gordian Knot is a 2D platformer developed in Unity using C# Coding within a 6 week time frame. For this project I wanted to try developing my first game so I wanted to keep it simple with a small 2D platformer. For this project I focused on the myth of the Gordian Knot in which Alexander the Great comes to a small town and cuts a knot in half, fulfilling a prophecy spoken by an oracle that we would go onto conquer all of Asia.
